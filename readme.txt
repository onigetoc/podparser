oEmbed, Embed and share your favorites codes editors, html and javascripts editors in Post, Page or in text Widget like Github Gist, Codepen, jsfiddle, Pastebin, JSbin, Snipplr, plnkr, Codeply, Ideone, Snip2code.

== Description ==

This plugin help you easly embed and share your favorites codes from the most populars codes editors, html. css and javascripts editors in Post, Page or in text Widget like Github Gist, Codepen, jsfiddle, Pastebin, JSbin, Snipplr, plnkr, Codeply, Ideone, Snip2code..

> ### Features: ###
> *   Embed code for the most popular codes editors website like: Github Gist, Codepen, jsfiddle, Pastebin, JSbin, Snipplr, plnkr, Codeply, Ideone, Snip2code
> *   Embed in text widgets
> *   Just enter url in posts, pages or text widget with space or return and it will embed automatically or use the embed shortcode. [embed]https://codepen.io/onigetoc/pen/YNdYBw[/embed]

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload 'codes_embed' to the '/wp-content/plugins/' directory
2. Unzip the file
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Just enter any urls in your Posts, Pages or text Widget anywhere in your content (You should add return space before and after the url )
